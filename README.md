# Eye Exercise

React + TypeScript + Vite app for webcam-based eye exercise tracking and guidance.

## Project Structure
- `src/` application code
- `dist/` production build output

## Run Locally
```bash
npm install
npm run dev
```

Open the app from the Vite URL (usually `http://localhost:5173`).

## Quality Checks
```bash
npm run lint
npm run build
```
